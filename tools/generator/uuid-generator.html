UUID 生成器

生成 UUID v4 / v7 与 ULID，加密级随机、纯本地运行

-

UUID v4

完全随机的 128 位标识符，格式 xxxxxxxx-xxxx-4xxx-yxxx-xxxxxxxxxxxx，适合绝大多数场景。

UUID v7

前 48 位为毫秒时间戳，可按生成时间排序，适合用作数据库主键以减少索引碎片。

ULID

26 字符 Crockford Base32 编码，时间有序、可排序且 URL 安全，比 UUID 更紧凑。

常见问题

UUID 生成器安全吗？生成的 UUID 会发送到服务器吗？

完全安全。所有 UUID 和 ULID 均在你的浏览器本地生成，使用 Web Crypto API 的加密级随机数，生成结果不会发送到任何服务器。

UUID v4、UUID v7 和 ULID 有什么区别？我该选哪种？

UUID v4 完全随机，适合绝大多数场景；UUID v7 前 48 位为毫秒时间戳，可按时间排序，适合数据库主键以减少索引碎片；ULID 同样时间有序，采用 26 字符 Base32 编码，比 UUID 更紧凑且 URL 安全。

生成的 UUID 会重复吗？

UUID v4 有 2^122 种可能值，碰撞概率极低，实际使用中可视为唯一。UUID v7 和 ULID 结合时间戳与随机部分，在同一毫秒内也极难碰撞。

如何批量生成多个 UUID？

在「数量」输入框中填入 2 到 100 之间的数字，点击「生成」按钮，工具会在文本区域中输出对应数量的 UUID，每行一个，可一键复制全部。

UUID 生成器可以离线使用吗？

可以。本工具是单文件 HTML，将页面保存到本地后，即使断开网络也能正常生成 UUID 和 ULID，无需安装或注册。
